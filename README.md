# SP-Sub
Submission repository for "Power of Words": Stealthy and Adaptive Private Information Elicitation via LLM Communication Strategies

# File Structure
- prompt_implementation.md: the prompt implementation for the estimation of user motivation and capability.
- subjective_evaluation.md: subjective evaluation entries for the evaluation study.
- baseline_condition.md: example of baseline condition.
- private_definition.md: private information definitions.
